# Broad iOS [![Build Status](https://travis-ci.org/carlossless/broad-ios.svg?branch=master)](https://travis-ci.org/carlossless/broad-ios)

Stream Live Broadcasts from [LRT](http://lrt.lt)

## Bootstrapping

```
brew bundle
bundler install
carthage bootstrap --platform ios
```

## Building the IPA

```
fastlane build
```
